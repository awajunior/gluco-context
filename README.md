# Gluco Context

**Independent diabetes context organizer for manual review and personal data workflows.**

Gluco Context helps users organize meal, glucose, insulin-on-board (IOB), sensor, profile and personal health context in one place.

It is designed to support personal review, structured snapshots and structured text prepared for optional user-directed use with an external AI service chosen by the user.

---

## What it does

- Reads compatible Nightscout data
- Organizes meal information and personal context
- Creates structured snapshots for manual review
- Supports optional reassessment reminders
- Stores user notes and selected actions locally
- Includes an optional home-screen widget
- Can open a user-configured compatible companion app

---

## What it does not do

Gluco Context is **not a medical device**.

It does not diagnose, treat, prevent, prescribe, calculate, recommend, deliver or automate insulin or any therapy.

All health and treatment decisions remain with the user and their healthcare team.

---

## Compatibility

Gluco Context is designed for personal workflows that may use Nightscout and compatible diabetes tools.

It is an independent project and is **not affiliated with, endorsed by, or distributed by** AndroidAPS, AAPS, Nightscout, iAPS or any other third-party project.

---

## Installation

Gluco Context is distributed as a sideload APK. It is not available on the Google Play Store.

Download the latest APK from the [Releases](../../releases) page and follow the instructions in [docs/INSTALL.md](docs/INSTALL.md).

**Always verify the SHA-256 hash before installing.**

## Privacy

Gluco Context does not collect, transmit or store any data on external servers.

All data remains on the user's device. The Nightscout URL and access token are entered by the user and never leave the device.

See [docs/PRIVACY.md](docs/PRIVACY.md) for full details.
